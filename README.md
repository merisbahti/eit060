Bygginstruktioner
=================

För att bygga: 

    $ ant

Körinstruktioner
================

Innan du kör programmen så måste du befinna dig i bin/.

Navigera till bin:

    $ cd bin
  
För att köra server:

    $ java server port
  
för att köra klient:

    $ java client ip port

där ip är localhost (om du kör lokalt).
